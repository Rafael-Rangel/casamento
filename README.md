# Casamento 12/12/2026 — Plano Financeiro

Planejador financeiro focado no casamento. A sobra do mês (salário + projetos − vida/cartão) alimenta o cronograma Junho → Dezembro.

## Como rodar

```bash
npm install
npm run dev
```

## O que tem

- **Casamento** — cronograma mensal, checkboxes, déficit e resumo (já pago / a pagar)
- **Dashboard** — visão geral de receitas, gastos e patrimônio
- **Salários / Projetos** — Gênesis e KoruVision aumentam a sobra
- **Vida & cartão** — despesas pessoais e faturas reduzem a sobra do casamento
- **Linha do tempo** — projeção mês a mês

Dados salvos no `localStorage` do navegador.
